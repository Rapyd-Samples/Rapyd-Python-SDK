# HostedPageStatus

Status of the hosted page. One of the following: NEW - The hosted page was created. DON - Done. The card was added to the customer profile. EXP - The hosted page expired.

**Properties**

| Name | Type | Required | Description |
| :--- | :--- | :------- | :---------- |
| NEW  | str  | ✅       | "NEW"       |
| DON  | str  | ✅       | "DON"       |
| EXP  | str  | ✅       | "EXP"       |
